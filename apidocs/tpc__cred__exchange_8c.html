Perform an RFC 8693 token-exchange request.

Sends a POST to the configured token endpoint with: grant_type=urn:ietf:params:oauth:grant-type:token-exchange subject_token=<JWT> resource=<source_url> audience=<source_url> scope=<scope>

Uses a fork/exec'd curl subprocess (same pattern as tpc_curl.c).
