1/*
2 * exchange.h — RFC 8693 OAuth2 token-exchange client (public API).
3 *
4 * WHAT: Declares brix_token_exchange(), which trades a subject access token
5 * for a new access token scoped/audienced for a backend origin.
6 * WHY: Phase-70 full credential delegation §5.4 — when a backend audience is
7 * node-bound the cache must mint an origin-specific token rather than
8 * replay the client's token verbatim.
9 * HOW: brix_token_exchange() POSTs an RFC 8693 grant to the configured OAuth2
10 * token endpoint and returns the minted access_token allocated from the
11 * caller's pool. All parameters are ngx_str_t; nothing is logged that
12 * could leak a token or the client secret.
13 */
