1/* http_common.h — unified brix storage/namespace directives (HTTP plane)
2 *
3 * WHAT: one module owns the bare storage grammar (brix_export,
4 * brix_storage_backend, brix_storage_credential, brix_cache_*,
5 * brix_stage*, brix_thread_pool, brix_cache_verify, brix_allow_write,
6 * brix_read_only, brix_compress) so every brix HTTP protocol shares a
7 * single directive surface.
8 * WHY: nginx's ngx_conf_handler is first-module-wins on directive names,
9 * so a shared name must be registered by exactly one http module.
10 * HOW: values land in this module's ngx_http_brix_shared_conf_t; protocol
11 * modules copy the merged values into their embedded `common` via
12 * brix_http_common_adopt() at merge_loc_conf time. Module emission
13 * order in ./config puts this module before the protocol modules, so
14 * its merge for a given location always precedes theirs.
15 */
