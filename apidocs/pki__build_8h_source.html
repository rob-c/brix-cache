17/*
18 * brix_build_ca_store — build an X509_STORE for CA/CRL-based certificate
19 * verification.
20 *
21 * @log: nginx log context.
22 * @cadir: directory of CA certificates; NULL to skip.
23 * @cafile: single CA bundle file; NULL to skip.
24 * If both cadir and cafile are NULL, system default paths are
25 * loaded via X509_STORE_set_default_paths().
26 * @crl_path: CRL file or directory path; NULL to disable revocation
27 * checking. When non-NULL the path must be stat()-able;
28 * failure is fatal (returns NULL).
29 * @extra_flags: additional X509_V_FLAG_* flags applied to the store before
30 * CA loading (e.g. X509_V_FLAG_ALLOW_PROXY_CERTS for GSI
31 * proxy certificate chains). Pass 0 for no extra flags.
32 * @crl_count_out: if non-NULL, receives the total count of CRLs loaded.
33 * @sp_mode: signing_policy enforcement mode (BRIX_SP_MODE_*). When a
34 * cadir is given its <hash>.signing_policy files are compiled
35 * and attached to the store. BRIX_SP_MODE_REQUIRE with only a
36 * cafile (no directory to search) is a configuration error and
37 * returns NULL.
38 * @crl_mode: CRL strictness (BRIX_CRL_MODE_*). Gates whether CRL verify
39 * flags are set and whether a missing CRL is tolerated (TRY)
40 * or fatal (REQUIRE).
41 *
42 * Returns a new X509_STORE on success, NULL on failure. On failure the store
43 * has been freed and no cleanup is required by the caller.
44 */
