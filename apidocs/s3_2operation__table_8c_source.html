1/*
2 * operation_table.c — S3 HTTP operation descriptor table.
3 *
4 * Lists every HTTP method handled by the S3 module with its metric slot and
5 * capability flags. The shared helpers in compat/protocol_caps.c use this
6 * table for method-slot classification and for the S3 OPTIONS response.

8 * Note: S3 is a REST API, not a WebDAV surface. There is no Allow header
