2 * wire_codec_file.c — file-family wire-body codec. See wire_codec.h.

4 * open / close / read / pgread / write / pgwrite / sync / truncate / writev /
5 * clone / chkpoint request bodies. These carry the hot-path multi-byte integer

7 * lived as scattered htonl/ntohl on both sides now lives here once. Pure C.

16xrdw_open_req_pack(const xrdw_open_req_t *r, uint8_t body[XRDW_BODY_LEN])
17{
18 if (r == NULL || body == NULL) {

22 xrdw_put_u16(body + 0, r->mode);
