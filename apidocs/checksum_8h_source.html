82 *
83 * WHAT: Reads entire file at fd in 64KB chunks, accumulates checksum into *out.
84 * Supports Adler-32, CRC-32 (zlib), and CRC-32c (table lookup).

86 * WHY: XRootD kXR_fattr responses include checksums; WebDAV PROPFIND returns checksum

89 * HOW: Loop pread with EINTR retry. Adler-32 uses adler32(), CRC-32 uses crc32(),

95/*
96 * brix_checksum_u64_fd - compute 64-bit checksum (CRC-64/XZ or /NVME) from fd.
97 *
98 * WHAT: Streams the whole file at fd and accumulates the selected CRC64 variant
99 * into *out. Rejects non-CRC64 algorithms (NGX_ERROR).
100 *
101 * WHY: S3 x-amz-checksum-crc64nvme, root:// kXR_Qcksum crc64, and WebDAV Digest

105 * HOW: Validates is_u64(alg), delegates to brix_cksum_u64_fd((int)alg, fd, out)

107 * read-error helper on failure.
