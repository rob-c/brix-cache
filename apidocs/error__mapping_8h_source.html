2 * error_mapping.h — unified errno → protocol status mapping.
3 *
4 * Consolidates three domains previously split across kxr_errno.h, http_errno.h,
5 * and result_mapper.h:
6 * 1. POSIX errno → XRootD kXR codes (brix_kxr_from_errno)
7 * 2. POSIX errno → HTTP status codes (brix_http_errno_to_status)
8 * 3. Namespace service status → HTTP (brix_http_map_ns_status, brix_http_map_errno)
9 */

17/*
18 * Sections 1-2 are ngx-free (pure uint16_t/int) and compile into the standalone

20 * module-only. When building libxrdproto (-DXRDPROTO_NO_NGX) we pull only the
21 * ngx-free enum (ns_status.h) and omit Section 3; the module build pulls the full
