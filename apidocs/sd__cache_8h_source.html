4/*
5 * sd_cache.h - the generic read-cache decorator (phase-64 section 12.1).
6 *
7 * WHAT: An SD-driver instance that WRAPS a source backend (the tier below: a stage
8 * decorator or the backend itself) and a cache store (a cstore adapter over
9 * any driver), interposing the READ-open path: a complete cached object is
10 * served from the cache store, a miss is filled from the source into the
11 * store and recorded in a cinfo. Every write / namespace / xattr / staged op
12 * delegates straight to the source (a write also invalidates the cached
13 * copy), and a served read object is the STORE's own object - so reads bypass
14 * the decorator entirely.
15 *
16 * WHY: Before phase-64 the read cache was a scheme-dispatched subsystem bolted to
17 * a local POSIX tree. Folding it into one composable decorator the registry
18 * stacks (cache(stage(backend))) gives ONE read cache that fronts ANY source
19 * through the cstore, with no driver/protocol branch above the SD seam (P3,
20 * review gate G5). The VFS resolves to the composed top and never knows a
21 * cache is present (G4).
22 *
23 * HOW: open(READ) consults cstore_cinfo_load: a COMPLETE hit returns
24 * cstore_serve_open; a miss runs the fill spine (source open/pread ->
25 * cstore_fill_*) then records the cinfo and serves. open(WRITE) passes
26 * through to the source and evicts the cached copy. The decorator's own
27 * byte/dir slots are never reached (open returns source/store objects).
28 *
29 * SP1 ships the decorator with an INLINE fill - correct for the WebDAV / S3
30 * worker-thread context, and not yet on the live root:// event loop (the
31 * config -> tier-grammar -> registry wiring is the deferred migration, and
32 * the non-blocking root:// async-offload + remote-store path is the SP2
33 * "shell -> full" step). The nearline (tape) recall+park branch is wired to
34 * the source's recall slot but only exercised once the frm driver lands
35 * (SP5). See docs/refactor/phase-64-fully-tiered-composable-storage.md
36 * (section 9, 10, 12, Appendix J).
37 */
