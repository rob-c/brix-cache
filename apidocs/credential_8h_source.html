4/* ---- Module: TPC Delegated Credential Model ----
5 *
6 * WHAT: Protocol-neutral representation of the credential a third-party copy
7 * carries to authenticate against the remote endpoint. Defines
8 * brix_tpc_credential_type_t (NONE / PROXY / TOKEN), the
9 * brix_tpc_credential_t value object (proxy_pem, bearer, identity,
10 * expires_at), and the parse/validate/type-name entry points implemented
11 * in credential.c.
12 *
13 * WHY: Both the stream (root://) and WebDAV (davs://) TPC paths must accept the
14 * same delegated-credential forms — a GSI proxy PEM or a WLCG bearer token
15 * — yet historically each transport parsed credentials its own way. A
16 * single typed model lets the shared TPC core normalise the credential
17 * once and reason about expiry uniformly.
18 *
19 * HOW: brix_tpc_credential_parse() sniffs the raw credential (honouring an
20 * optional caller hint, the "Bearer " prefix, or a "-----BEGIN" PEM
21 * marker) into a typed brix_tpc_credential_t; brix_tpc_credential_validate()
22 * rejects empty or expired credentials; brix_tpc_credential_type_name()
23 * maps the enum to a stable log/metric string.
24 */

45/* Sniff a raw credential into *cred (zeroed first), classifying it via the
46 * caller hint or the "Bearer "/"-----BEGIN" markers after trimming surrounding
47 * whitespace/CRLF; the "Bearer " prefix is stripped from tokens. Empty/NULL
48 * input or the literal "none" yields type NONE. When pool is non-NULL the value
49 * is copied NUL-terminated into it; when pool is NULL *cred aliases the caller's
50 * bytes (borrowed, must outlive cred). Returns NGX_OK, NGX_DECLINED for an
51 * unrecognised format, or NGX_ERROR on NULL cred / allocation failure. */
