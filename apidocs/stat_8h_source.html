25/* ---- Function: brix_cache_path_flag() ----
26 *
27 * WHAT: Checks whether a requested path exists in the local cache (conf->cache_root). Returns kXR_cachersp
28 * flag if found, 0 otherwise. Tells clients the file is served from cache rather than origin.

30 * WHY: Clients can optimize read patterns when they know content is cached locally — skip origin fetches for
