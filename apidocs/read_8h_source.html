75/* ---- Function: brix_pgread_read_encode_inplace() ----
76 * Zero-copy paged read: reads file data DIRECTLY into the final kXR page-mode
77 * wire buffer `out` ([CRC32c(4)][data] per page, file-offset aligned) via batched
78 * preadv and computes each page CRC32c in place — no flat-buffer copy. `out` must
79 * hold rlen + ceil-pages * 4 bytes. Returns encoded byte count; sets io->nread to
80 * bytes read (-1 on I/O error, io->io_errno = errno). Output is byte-identical to
81 * brix_pgread_encode_pages over the same bytes. Safe to call on a worker thread
82 * (pure I/O + CRC; touches no ctx/connection/pool).
83 *
84 * When io->nowait is set the batched reads use preadv2(RWF_NOWAIT): if the whole
85 * range is NOT page-cache resident (any short batch or EAGAIN) the call aborts
86 * early with io->nread=0 and io->io_errno=EAGAIN, having produced nothing
87 * usable — the caller must treat that as a miss and fall back to a blocking
88 * read. A clean hit returns the full encoding with io->nread==rlen. This powers
89 * the event-loop warm-cache fast path (skip the thread-pool offload when the
90 * data is resident); leave nowait 0 for the normal blocking read.
